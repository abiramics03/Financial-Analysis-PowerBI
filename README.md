# Financial Analysis and Profitability Dashboard

## Project Overview
This Power BI project analyzes financial performance using General Ledger data.
It focuses on Revenue, Expenses, Cash Flow, and Profitability.

## Objectives
- Analyze revenue and expenses
- Track inflow and outflow
- Understand net cash flow
- Build interactive financial dashboards

## Tools Used
- Power BI
- DAX
- Excel

## Key Features
- KPI Cards
- Bar, Line, Pie, Waterfall charts
- Slicers (Year, Month)
- Cash Flow Analysis

## Outcome
The dashboard helps understand financial health and supports decision making.
